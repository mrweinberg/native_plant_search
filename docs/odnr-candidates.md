## Context

This document ranks each missing species by relevance for inclusion. Trait data (light, soil, bloom, native states, height, etc.) can be hydrated later from authoritative sources (Lady Bird Johnson Wildflower Center, Missouri Botanical Garden, USDA PLANTS) once the canonical list is locked.
